BINNA Final Project

Open index.html in your browser.

Structure:
- customer/: landing, login, signup, dashboard, marketplace, cart, checkout, orders, AI reports, settings.
- supplier/: supplier overview, orders, order details, inventory, payments, store settings.
- admin/: dashboard, users, suppliers, add supplier, add supplier product, categories, orders, order details, disputes, analytics, settings.
- assets/: BINNA logo.
- tailwind.config.js: single Tailwind CDN configuration file used by all pages.
- script.js: shared small interactions.

Notes:
- All HTML pages use Tailwind CSS CDN.
- There are no CSS files in this project.
